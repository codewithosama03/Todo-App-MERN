Frontend (React UI)
        ↓ (HTTP request)
Backend (Node + Express API)
        ↓
Controller (logic)
        ↓
Model (schema)
        ↓
Database (MongoDB Atlas)


1. Request comes from frontend/Postman
        ↓
2. server.js → sees /tasks
        ↓
3. routes/taskRoutes.js
        ↓
4. controller (createTask)
        ↓
5. model (Task)
        ↓
6. MongoDB stores data
        ↓
7. Response sent back


Routes → WHERE
Controllers → WHAT
Models → HOW DATA LOOKS
Server → CONNECTS EVERYTHING
Database → STORES DATA
